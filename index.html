Move Soot

Use up and down arrows to move forwards and backwards
Use left key to turn left
Use right key to turn right

Move Camera

Use w, a, s, and d to move
Use space key to go up and shift to go down
Use q and e to rotate left and right
Use m to switch to and from mouse control

Edit World

Use f to set place block mode
Use v to set delete block mode
Use t to escape edit mode
Use button for stone
Click to place and destroy blocks that are highlighted brown
